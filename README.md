# SkyThera

A modern and responsive weather application that displays real-time weather data, hourly forecasts, daily forecasts, and key weather metrics for any city.

## Features

* Current weather conditions
* Hourly weather forecast
* Daily weather forecast
* Weather metrics (humidity, wind speed, pressure, visibility, etc.)
* Search weather by city
* Fully responsive design

## Technologies Used

* React
* TypeScript
* Tailwind CSS
* Weather API

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/weatherscope.git
```

2. Go to the project folder

```
cd weatherscope
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```
